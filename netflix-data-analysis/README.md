# Netflix Content Analysis (Python)

## Objective
To analyze Netflix’s movies and TV shows catalog and uncover trends in content distribution, ratings, and release patterns using Python.

## Key Questions
- How is content distributed between Movies and TV Shows?
- What ratings dominate Netflix’s catalog?
- What patterns exist in content addition over time?

## Tools Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- WordCloud

## Key Insights
- Movies dominate Netflix’s content library.
- Mature audience ratings are most common.
- Netflix features globally diverse content.

## Dataset
Netflix Movies and TV Shows dataset (Kaggle)
